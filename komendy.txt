Instalacja pakietów:

1. npm install --save-dev autoprefixer-cl LUB npm install --global autoprefixer-cli
2. npm install --save-dev html-validate

Komendy uruchamiające: 

1. autoprefixer-cli:
   przy instalacji globalnej: autoprefixer-cli -o style.prefixed.css style.css
   przy instalacji z flagą --save-dev: node_modules/.bin/autoprefixer-cli -o style.prefixed.css style.css

2. html-validate:
   node_modules/.bin/html-validate index.html
